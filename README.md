# IoT Based Patient Health Monitoring System Using ESP32

## Overview

This project is an IoT-based patient health monitoring system developed using the ESP32 microcontroller. The system collects real-time health data using multiple sensors and displays the information through a web server and IoT platform for remote monitoring. It is designed to support continuous patient observation in home care and remote healthcare environments. :contentReference[oaicite:1]{index=1}

## Features

- Real-time patient health monitoring
- ESP32-based wireless data processing
- Web server dashboard for live data visualization
- Mobile monitoring using Blynk IoT platform
- Body temperature measurement
- Room temperature and humidity monitoring
- ECG signal monitoring and heart rate estimation
- LED-based abnormal temperature alert system

## Hardware Components

- ESP32 Microcontroller
- DHT11 Temperature and Humidity Sensor
- DS18B20 Body Temperature Sensor
- AD8232 ECG Sensor
- LED Indicator
- Breadboard and connecting wires

## Working Principle

The sensors continuously collect health-related data and send it to the ESP32 microcontroller. The ESP32 processes the data and transfers it through Wi-Fi to the Blynk server and a locally hosted web server. Users can monitor temperature, humidity, ECG signals, and heart rate through the dashboard interface. :contentReference[oaicite:2]{index=2}
